## ğŸ“– Usage

### Creating Timers

1. **Standard Timer**: Click the `+` button â†’ Select preset or create custom
2. **Stopwatch**: Click the stopwatch button â†’ Start counting up
3. **Pomodoro**: Select Pomodoro mode â†’ Choose focus/break type

### Stack Timers

1. Click the **Stack** button (top-right corner)
2. Enter stack name and toggle recurring if needed
3. Add phases with name, duration, and description
4. Preview timeline on the right
5. Click **Create Stack** to save
6. Find saved stacks in the left sidebar
7. Click **Run Stack** to execute

### Keyboard Shortcuts

- **Space** - Play/pause active timer
- **R** - Reset active timer

### Fullscreen Modes

- **Timer Fullscreen** - Click maximize icon on timer card
- **Clock Fullscreen** - Click clock in top-right corner
